# Personal Computer Setup - Windows 11

I don't typically use Windows, but if I give it a go - this guide will mostly deal with removing bloat-ware
and figuring out how to set up all the good stuff like: git, node, docker, SSH, python, hypervisors, local DNS, etc.

Also include a list of must-have software i.e. VLC, multiple archive format support, terminal, fonts, filesystem support.

Also guides on securing the system.

Also guides on configuring Windows appearance / behaviours / menus and widgets.
